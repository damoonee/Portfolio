Portfolio de Sophie Bluel

Ce projet à pour but de travailler sur la conception du site portfolio d'une architecte d'intérieur. En utilisant un backend déjà créé, j'ai importer dynamiquement avec JavaScript les projets réalisés par le designer d'intérieur, avec des filtres par catégories. J'ai également créé la page de connexion en faisant appel a une API avec une redirection vers la page d'acceuil lorsque l'administrateur est connecté avec succès. Pour finir, j'ai implémenté une modale sur la page d'acceuil, permettant a l'admin d'ajouter ou supprimer un travail dynamiquement.
